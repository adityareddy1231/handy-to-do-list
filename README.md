# Handy To-Do list

A simple but handy To-Do list application built using HTML, CSS, javaScript and jQuery.

You can check it out here: https://adityareddy1231.github.io/handy-to-do-list/

Instructions:
Click on the plus sign to show the input box.
Type your to-do item and hit "Enter" to add it to list.
You can click on the added items to strike-through completed ones.
Re-clicking will remove the strike-through and make it normal.
You can also delete the to-do item completely by clicking on the trash icon.
